## ⚡ Быстрый старт

1. Склонируйте репозиторий:
```bash
git clone https://github.com/ВАШ_НИК/AI_Debuger.git
cd AI_Debuger/backend
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Запустите сервер:
```bash
python main.py
```

4. Откройте в браузере:
```
5. **http://127.0.0.1:8000**
```
